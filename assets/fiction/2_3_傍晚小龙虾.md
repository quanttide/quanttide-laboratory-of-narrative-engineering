# 傍晚小龙虾

“你今天做的很棒。我很羡慕这样的你。”

从十年前开始到现在。陆知微心里默默对自己说。

“被这些人搞烦了，看到这些人说这些话就应激反应。”林远亭一边剥着小龙虾壳一边说。

林远亭看陆知微似乎在想什么，有点不知道是该等她开口还是继续说下去。

陆知微看着林远亭关切的眼神，突然涌上一股倾诉欲。

她喜欢了他这么多年，第一次亲眼看到他这么关心地看她。她实在忍不住。

“没关系的，我觉得我应该还算是个不错的倾听者。”林远亭补充道。

不忍了。

“我羡慕你，是因为我想做但是做不到。”

“最近有个被投的团队有纠纷。我想按定下来的条款执行退出，但是有点狠不下来心。”

她握着小龙虾的手顿了顿。

“我总觉得，这是一种放弃。”

林远亭笑了，“我前一阵子也这样在折磨自己。”

“团队开始大了起来，以前靠我当大家长的模式维系不下去了，我的心也累了。”

“但是规则很难建立。得一次一次地提醒自己，必须要狠下心来，按着所有人按规则走。”

她也笑了。“你可真是个狠人，对别人狠，对自己更狠。一定很难吧。”

看着她收住的笑容和心疼的眼神，他也忍不住了。

他想起第一次看见她的惊艳。十年后的这一刻，惊艳更甚与当时。

时间似乎没有在她身上留下过什么痕迹。她还是那么纯粹和热烈。

“难。想这些事情焦心，整夜整夜地睡不着。”

“有一种把自己内心挖空的感觉。那一阵子感觉自己和世界都隔离开了，每天都在处理这样那样的糟心事。”

看她要哭的样子，他赶忙逗她，“要不要挖出来给你尝一口？”

“不行不行！”她赶忙摇头。

“我觉得我最终还是会下决心退。不过得先过了心里这关。”她一边拆着龙虾壳一边说。

“我们家里的小龙虾比这个头还要大点。有好一阵子没回去了。”他又拿起一只小龙虾。

一盆小龙虾很快见底。两个人都饱了，但似乎都还没有尽兴。

陆知微伸手喊老板结账。

老板在算账的时候，林远亭夸道，“你家饭菜份量真足。”

老板听到这句话，一激动，话匣子打开了，“我们最怕遇到客人说分量少。打工的客人吃的多饭量大，不像那些游手好闲的。以前我们出去打工，吃的也多。现在在这上班，吃的少多了。”

他们认真地听着老板的滔滔不绝。

结账完，他们一起走在回去的路上。

她说，“听到老板说这算上班的时候我有点恍惚。原来打工和上班在不同的人眼里是那么不同。”

他说，“我很喜欢和店主们聊天。他们的倾诉欲好多都很强，能从他们身上听到很多故事。”

她说，“如果能把这些故事都记录下来就好了。很希望他们被看到。”

他说，“我很喜欢他们。我觉得比精英圈子里那群道貌岸然的精英有意思地多。”

她说，“现在的你，身上一股土洋混合的感觉。一边说着泥土里一样的大白话，一边又说着横跨 N 个学科没人跟得上的概念。给人一种莫名其妙的混搭感。”

“我觉得我和你在一起的时候，很容易回到这种不土不洋的原生态。和你在一起很放松。”他躲开她的视线看向远方。

她的心里被戳了一下。

“我喜欢观察这些餐馆的变化。看着一些餐馆倒下，另一些餐馆取而代之，看着市场的变迁，感知它的脉络。无数次被他们安抚焦躁的内心。”

所以，他是经历了多少焦躁不安。她想了想，没问出口。

“大隐隐于市。这种感觉还蛮好的。”
