# 可产品化功能清单

从 10 个叙事工程实验中提取的可产品化功能，按 3R 工作流（Review → Reflect → Rewrite）组织。

## Review 评审

| 功能 | 实验 | 输入 | 输出 | 成熟度 |
|------|:---:|------|------|:---:|
| **作者审美评审** | p09 | scene + style.yaml | 11 维度评分 + 证据 + tension | 🟡 |
| **母题提取** | p05 | scene | 3-6 母题 + weight + evidence | 🟡 |
| **叙事结构标注** | p01 | scene | 段落级起承转合标注 | 🟢 |
| **盲品归因** | p05 | 多场景母题 profile | 场景系列分类（同系列聚类率） | 🟡 |

## Reflect 反思

| 功能 | 实验 | 输入 | 输出 | 成熟度 |
|------|:---:|------|------|:---:|
| **母题缝隙分析** | p08 | scene + motif.yaml | 缺口报告 + 6 方向改进建议 | 🟢 |
| **风格-母题根因诊断** | p10 | scene + style.yaml + motif.yaml | 弱维度→缺失母题映射 | 🟢 |
| **多向改进建议** | p08 | gap report | 增强/引入/借用/转化/克制/反向 6 条改法 | 🟢 |
| **风格一致性监控** | p09 | 系列多场景 | 维度评分趋势 + 偏离检测 | 🟡 |

## Rewrite 重写

| 功能 | 实验 | 输入 | 输出 | 成熟度 |
|------|:---:|------|------|:---:|
| **母题约束生成** | p07 | scene template + motif set | 带母题约束的场景文本 | 🟢 |
| **跨作品母题借用** | p06 | target motif + source series | 跨作品母题变体建议 | 🟡 |
| **组合改法生成** | p10 | scene + style + motif | 风格根因 → 母题引入的具体改法 | 🟢 |

## 知识库建设

| 功能 | 实验 | 输入 | 输出 | 成熟度 |
|------|:---:|------|------|:---:|
| **母题跨作品网络** | p06 | 两作品场景对 | 跨作品母题映射 + 相似度 | 🟡 |
| **风格-母题映射学习** | p10 | 作者成稿 + style.yaml + motif.yaml | 个人"风格↔母题"映射矩阵 | 🟡 |
| **审美配置文件** | p09 | author style.yaml | 可复用的审美 Agent | 🟢 |

## 优先封装建议

| 优先级 | 功能 | 原因 |
|:---:|------|------|
| 🔴 | **母题提取** (p05) | p10/p08 的前置依赖，所有下游功能都依赖提取结果 |
| 🔴 | **母题缝隙分析** (p08) | 完整的诊断→建议管线，可直接作为 Reflect 环节 |
| 🟡 | **作者审美评审** (p09) | style.yaml 已有完整数据，API 化即可用 |
| 🟡 | **组合改法生成** (p10) | 效果最好的改法模式，但依赖 p05+p09 前置 |
| 🟢 | **母题约束生成** (p07) | Rewrite 环节的核心能力 |

---

**成熟度说明**：🟢 实验验证稳定 / 🟡 需进一步校准 / 🔴 仅概念验证
