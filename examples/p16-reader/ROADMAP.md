## 立即执行 (P0)

### 任务 ①：E4-0 Prompt 操控性检验

**问题**：核心假设"LLM 能否通过 prompt 模拟分化读者"完全未经验证。这是 Stage-Gate 的第一道门，18 次调用即可给出 go/no-go。

**实施**：
1. 定义 P0–P5 六个画像的 prompt 模板（数值参数 + 行为锚定两套，后者作为 fallback）
2. 编写脚本：调用 LLM 生成读者自述 → embedding → 计算轮廓系数和画像间距
3. 结果写入 `results/e4-0_manipulation_check.json`

**预期影响**：如果通过（轮廓系数 ≥ 0.25），进入 E4-1。如果不通过，尝试行为锚定 prompt 重跑；仍不通过则项目终止。

**估计耗时**：2 天（含 prompt 调试）



## Phase I 执行 (P1)

> Phase I 的全部实验无层1依赖，成本可控（总调用量 ~168 次）。

### 实验 ③：E4-1 分化验证

**实施**：
1. 在 `src/phase1/` 下实现 prompt 模板系统
2. 5 画像 × 6 文本 × 5 次调用 = 150 次 LLM 调用
3. 分析：判别效度（4 组对比方向）、稳定效度（ICC）、客观基线（植入错误检出率）
4. 结果写入 `results/e4-1_differentiation.json`

**门控条件**：4 项验证全部通过 → E4-2。否则项目终止。

**估计耗时**：3 天（含 prompt 调试和结果分析）

### 实验 ④：E4-2 跨文本泛化与外部锚定

**实施**：
1. 利用 E4-1 的模拟数据 + E3 的历史输出
2. 3-fold CV 画像分类 + 与 E3 评分的相关性分析
3. 结果写入 `results/e4-2_generalization.json`

**门控条件**：全部通过 → Phase I 通过，进入 Phase II。未通过 → 项目降级为"纯 prompt 模拟读者"，不进入层1工程化。

**估计耗时**：1 天（数据分析为主）

## 探索性方向 (P3)

### 探索 ⑦：抽象母题的读者感知差异

**背景**：p07 发现抽象母题（孤独、歌声）在生成中约束力弱。一个可能的解释是：抽象母题在不同读者中的感知差异更大，因此"模拟读者"对它们的分化能力也更强（或更弱）。

**实验**：用 E4-1 的数据，比较"具象母题文本"与"抽象母题文本"上画像间的分化程度（效应量大小）。

### 探索 ⑨：temperature 对分化效度的影响

**问题**：当前固定 temperature=0.7。如果降低到 0.3，分化效应是否更稳定？但会不会降低 LLM 的角色扮演灵活度？

**实验**：用 P2 vs P5（预期分化最大的一对）在单篇文本上，以 temperature=0.3/0.5/0.7/1.0 各跑 5 次，比较 ICC 和分化效应量。

## 产品化路径

| 阶段 | 依赖 | 产出 |
|------|------|------|
| Phase I | E4-0/1/2 | "LLM 能否模拟读者"的确定性结论 + 纯 prompt 模拟读者原型 |
| Phase I+（降级路径） | E4-0/1/2 部分通过 | 纯 prompt 模拟读者作为实验工具（有结论：LLM 可模拟/不可模拟） |
| Phase II | Phase I ✅ + E4-3/4/5 | 含认知负荷层的可解释模拟读者引擎 |
| v1 | Phase II ✅ | 可配置画像 + 评价分布的 API |
| v2 | 探索 ⑦⑧⑨ | 经真实读者校准的置信区间 / 抽象母题感知 / temperature 策略 |

## 已知不推荐的方向

- ❌ **在 Phase I 通过之前实现层1**：核心假设未验证之前投入工程，违反本实验的两阶段门控原则
- ❌ **自然文本客观基线**：文本中自然的"结构异常"正是画像差异的一部分，不是真正的客观项
- ⚠️ **真实读者招募作为 Phase I 依赖**："LLM 模拟读者"和"真实读者"是两个独立问题，先验证前者，后者是可选增强
- ❌ **多模型对比**：当前设计使用单一 LLM，目的是最小化变量。模型切换实验应放在 Phase III
