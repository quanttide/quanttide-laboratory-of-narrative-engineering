# 叙事工程实验室 — 概念验证清单

验证叙事工程的核心假设：**写作能否被系统性地分析、度量和改进。**

---

## 已完成的验证

| PoC | 结论 | 状态 |
|-----|------|------|
| **p01** 叙事框架对比 | 起承转合、三幕、情感曲线弧线对齐。起承转合粒度适中，适合作默认框架 | ✅ |
| **p03** 风格可提取性 | 同作者聚类 100%、归因 100%。StyleStore 策略成立 | ✅ |
| **p04** Review 可靠性 | 体裁判断 15/15 一致，聚类 100%。意图理解稳定 | ✅ |
| **p02** 3R 效果度量 | Reflect 多角度归因可行，88-92% 空隙为无意识忽略。评分必须脱离链条 | ✅ |

---

## 3R 最终定义

```
Review:  理解文本意图     → {genre, intent, stage, summary}
Reflect: 检测空隙+多角度归因 → [{gap_type, 结构/人物/读者/技法归因}]
Rewrite: 带理解去修改     → 新版本
```

---

## 探索到实施的缺口

### 高风险（假设不成立则实施方向要改）

| 缺口 | 原因 |
|------|------|
| **Reflect 对人类作者是否有用** | 全部实验是 AI→AI 闭环。AI 的多视角归因对人类是启发还是噪音？未验证 |
| **独立评分模型不存在** | 评分必须脱离 3R 链条，但用什么模型、评什么维度、基线在哪？未探索 |
| **仅验证了小说体裁** | 案例库全是言情小说。起承转合对技术文档、营销文案、学术论文是否有效？未知 |

### 中风险（实施中遇到但能调整）

| 缺口 | 原因 |
|------|------|
| **Rewrite 情境引导生成未验证** | 修正版 p02 让 AI 直接改写，跳过了"生成引导→人类自改"的产品路径 |
| **短文本边界未知** | 第六章 6 段时三框架等价。1 段推文/广告语的文本，3R 是否还有意义？ |
| **阶段判断的"完成度光谱"未标定** | 当前仅二元判断（draft/final），产品是否需要更细粒度？ |

### 低风险（实施中处理即可）

| 缺口 | 原因 |
|------|------|
| **规则引擎 vs LLM 的空隙检测成本** | 规则检常规空隙 + LLM 检复杂空隙的混合方案待评估 |
| **非中文未测试** | 架构无语言假设，但未验证 |
