# Motif 母题分析管线 — 整合路线图

## 一、领域模型总览

```
Gallery (知识库)
├── Motif Ground Truth (motif.yaml, 三层)
│   ├── Shared [6] —— 手势先于语言/日常的缝隙/双向奔赴/温和的善良人/旁观者/时间
│   ├── Urban [5] —— 十年(10)/手势(9)/雨(7)/孤独(7)/歌声(5)
│   └── Campus [4] —— 论坛(10)/协作书写(9)/旁观者(8)/随身携带的温柔(6)
└── Style Framework (style.yaml, 11 维度)
    └── Dimension [11] —— 母题矛盾/视角/时间/语言/情感/细节/人物/对话/节奏/时代/美学

Experimental Pipeline
p05 Extract ──→ p06 Cross-Work ──→ p07 Consistency ──→ p08 Gap ──→ p10 Synthesize
  (LLM 提取)      (跨作品识别)        (约束验证)          (缝隙分析)    (风格改进)
```

### 核心实体关系

```
Article/Scene
├── Extracted Motif [3-6]          ← p05/p08/p10 提取
├── Variant [0..*]                 ← p06 跨作品变体（同一母题的不同表现）
├── Gap [0..*]                     ← p08 与 target profile 的偏差
│   ├── Covered（已覆盖）
│   ├── Missing（缺失）
│   └── Weak（弱化）
└── Style Review                   ← p10 风格诊断
    └── Diagnosis [0..*]           ← 弱维度 ↔ 缺失母题的关联

Suggestion [6 directions]          ← p08 改进建议，按方向分组：
    amplify / introduce / borrow / transform / restrain / reverse
```

### 通用语言

| 术语 | 定义 |
|------|------|
| **母题 (Motif)** | 叙事中反复出现的主题元素。与主题不同：主题是"说什么"，母题是"怎么说" |
| **变体 (Variant)** | 同一母题在不同作品/场景中的不同表现 |
| **缝隙 (Gap)** | 文章实际母题与目标母题库的偏差 |
| **吻合度 (Alignment)** | 约束生成后检测到的母题与目标的匹配程度（0-1） |
| **覆盖率 (Coverage)** | 提取/生成的母题占目标母题的比例 |
| **归因 (Attribution)** | 分析母题缺失的根本原因 |

## 二、实验管线现状

| 模块 | 核心问题 | 状态 | 关键发现 | 关键缺陷 |
|------|---------|------|---------|---------|
| **p05 提取** | LLM 能否从文本中提取母题？ | ✅ 合格 | 单篇覆盖 56-58%，联合 60-75%，盲品聚类率 79% | 覆盖率计算未去重；语义匹配用 LLM 导致循环论证；无人工校准 |
| **p06 跨作品** | LLM 能识别跨作品母题变体吗？ | ✅ 相似度可靠 ❌ 聚类失败 | 相似度矩阵 F1=100%（同 0.839 vs 异 0.133） | 配对盲测 0%，聚类 ARI≈0.25——单句摘要不足，无参照分类比有参照判定难得多 |
| **p07 一致性** | 母题约束能否保持场景连贯？ | 🟡 方向确认 | 约束组 > 对照组 +20~45%，约束组文本长 50%+ | keyword-map 同时高估和低估；抽象母题（孤独/歌声）约束力 0% |
| **p08 缝隙分析** | 能检测初稿缺失的母题并给建议吗？ | 🟡 实验已完成 | 6 方向建议框架可用，D1-D4 缝隙检测与预期一致 | 待评审：方向间多样性、人工评估、克制/反向方向的价值验证 |
| **p10 风格改进** | 母题诊断能否驱动更好的风格改进？ | 🟡 实验已完成 | 三组改法对比框架可用，LLM 自主发现 2-3 个风格-母题关联 | 待评审：组合改法是否真的优于单层、风格-母题映射的 Jaccard |

## 三、整合路线图

### Phase 1：可信化 (当前→Q3 2026)

修复已有实验中的方法论缺陷，确保评估指标可靠。

| 优先级 | 任务 | 涉及模块 | 说明 |
|--------|------|---------|------|
| P0 | **覆盖率去重** | p05 | `matched_count` 改为基于 unique GT 母题的计算，修复高估 15-28% |
| P0 | **语义匹配替代 keyword map** | p05, p07 | 引入 embedding 模型（text-embedding-3-small）替代 LLM 循环论证和手写 keyword 匹配 |
| P0 | **盲品平局处理** | p05 | 平局组按 50% 正确率计算 |
| P1 | **人工校准** | p05, p08 | 抽样验证 evidence 准确性、语义匹配对的人工确认 |
| P1 | **交叉验证检测管线** | p07 | 用 GPT-4o-mini 做交叉验证，消除同源偏差 |
| P1 | **完整段落版聚类** | p06 | 用 300-500 字完整场景段落替代单句摘要，重跑步骤 3/4 |

### Phase 2：系统化 (Q3→Q4 2026)

统一各模块的评估框架，建立可复用的基础设施。

| 优先级 | 任务 | 说明 |
|--------|------|------|
| P1 | **提取 prompt 统一管理** | 将内嵌在 py 文件中的 LLM prompt 提取到 `prompts/` 目录，实现跨 p05/p08/p10 复用 |
| P1 | **Gallery 知识库版本化** | motif.yaml/style.yaml 纳入版本管理，每次实验记录使用的 gallery 版本 |
| P1 | **实验报告标准化** | 统一 STATUS/README/ROADMAP 模板，输出机器可读的 JSON 格式报告 |
| P1 | **场景模板重新设计** | p07 为低覆盖率母题（歌声/协作书写/旁观者/孤独）设计专属场景模板 |
| P2 | **抽象母题翻译为可执行约束** | 将分析性母题定义改写为生成性行为指令（如"孤独"→"写一段内心独白"） |
| P2 | **同作品不同母题对照基线** | p06 在相似度矩阵中加入同作品不同母题的对，排除"风格元素"混淆 |

### Phase 3：产品化预备 (Q4 2026→Q1 2027)

验证产品化价值，封装核心能力。

| 优先级 | 任务 | 说明 |
|--------|------|------|
| P2 | **母题提取窗口敏感性分析** | p05：不同文本长度（1k/2k/3k/全文）的覆盖率饱和点 |
| P2 | **联合提取 ROI 分析** | p05：1/2/3/5 篇样本量下的覆盖率增长曲线 |
| P2 | **母题层级结构发现** | p06：对 14 个场景做层次聚类（dendrogram），探索 sub-motif 概念 |
| P2 | **约束力与文本质量相关性** | p07：20 篇文本做 2 人独立质量评分，检验吻合度是否 == 高质量 |
| P2 | **motif + style 联合约束实验** | p07：4 组对比（无约束/仅 motif/仅 style/motif+style） |
| P2 | **跨场景连贯性检验** | p07：5 个约束场景拼接，审查角色/时间线/性格一致性 |
| P2 | **跨作品母题检索原型** | p06：用 embedding + LLM 在 `assets/fiction/` 中构建"母题→段落"索引 |

## 四、已知方向性决策

### ✅ 已验证可行
- **pairwise 跨作品母题判定**（p06 相似度矩阵 F1=100%）——给定两个场景，LLM 精确判断是否同一母题
- **母题约束生成**（p07 约束组 > 对照组 +20~45%）——motif 作为生成约束有效，且提升文本质量
- **6 方向建议框架**（p08）——多方向改法在多样性上成立
- **风格-母题关联诊断**（p10）——LLM 能自主发现 5 个人工关联中的 2-3 个

### ❌ 已证明不可行
- **无监督母题发现**（p06 聚类 ARI≈0.25）——LLM 无法在盲测中自主重建 gallery 分类体系
- **单句摘要做母题匹配**（p06 配对盲测 0%）——粒度太粗，需要完整段落上下文
- **keyword-map 匹配**（p07）——同时 false positive 和 false negative 严重
- **抽象母题作为生成约束**（p07 "孤独"覆盖率 0%）——分析性概念不能被 LLM 直接"生成"

### ⚠️ 需进一步验证
- 组合改法 vs 单层改法的真实差异（p10 待评审）
- "克制"和"反向"方向的创作价值（p08 待评审）
- motif + style 联合约束是否 1+1 > 2
- 母题提取的文本窗口饱和点
