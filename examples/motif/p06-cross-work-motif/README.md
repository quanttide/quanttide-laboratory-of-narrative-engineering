# p06 — 母题跨作品识别实验

验证 LLM 是否能识别同一母题在不同作品中的不同表现形态，以及能否建立跨作品母题关联。

## 假设

1. 同一母题在不同作品中表现为不同"变体"（如"手势"在都市 = 毛巾，在校园 = 纸巾），但核心结构一致
2. LLM 可以识别这些变体属于同一母题，并给出跨作品映射
3. 该能力可用于构建跨系列母题网络，实现母题驱动的作品检索

## 背景

Gallery 中 `fiction/motif.yaml` 从都市言情和校园言情共同提炼出 6 个跨系列母题。这些母题是人工推导的——p06 的目标是用 LLM 盲测来验证这些跨作品关联是否具有客观基础。

例如，"手势先于语言"这个母题在都市言情中表现为"毛巾擦头发"，在校园言情中表现为"纸巾擦眼泪"和"披外套"。人类读者可以识别它们是同一叙事模式的不同实例，但 LLM 能做到吗？

## 实验步骤

```
步骤 1: 场景片段准备
  └─ 从 p05 步骤 1 产出的 8 篇 JSON 中提取每个母题的 evidence 字段
  └─ 将原文线索整理为独立的场景片段（每段 150-300 字）
  └─ 为每个片段标注：母题名、作品来源（都市/校园）、场景名
  └─ 输出：14 个场景片段（7 个母题 × 2 个变体）

步骤 2: 跨作品相似度矩阵
  └─ 两阶段策略（降低成本）：
      ├─ 阶段 1（embedding 预筛选）：对 14 个场景片段用 text-embedding-3
      │    生成稠密向量，计算 91 对余弦相似度矩阵作为 baseline
      ├─ 阶段 2（LLM 精准判定）：仅对 embedding 判定的"模糊边界对"
      │    （相似度 0.5-0.8 区间）做 LLM 判定，预期减少 50%+ 调用量
      └─ 同时保存 embedding 结果和 LLM 结果，对比两种方法的差异
  └─ LLM 判定提示词：
      "以下是从两部不同小说中提取的两个场景片段。
       请判断它们是否体现了同一叙事母题（recurring motif）。
       母题是贯穿叙事的重复性主题元素，而非单一情节事件。"
  └─ 输出格式：[{
        "pair": "手势_都市_毛巾 vs 手势_校园_纸巾",
        "same_motif": true/false,
        "similarity": 0.0-1.0,
        "shared_pattern": "如果有共同结构，用一句话描述",
        "reasoning": "判断理由"
      }]
  └─ 按"同一母题内 vs 不同母题间"分组计算平均相似度
  └─ 额外对照组：计算"同一作品不同母题间"的平均相似度（排除同作品风格
      元素的影响，用于检验母题相似度是否真的超越作品边界）

步骤 3: 母题链重构（开放探索）
  └─ 不使用 gallery 的母题标签，将 14 个场景片段直接提交给 LLM
  └─ 要求 LLM 将片段聚类，并为每个类命名母题
  └─ 提示词：
      "以下是 14 个来自两部小说的场景片段。
       请将它们按体现的母题分组成若干类。
       每个类内部应有相同的叙事模式或主题结构。"
  └─ 输出格式：{"clusters": [
        {"motif_name": "LLM 自行命名", "members": ["场景A", "场景D"]}
      ]}
  └─ 衡量：LLM 的聚类结果与人工标注的 7 个母题分类的吻合度（ARI 调整兰德指数）

步骤 4: 配对盲测
  └─ 随机抽取 4 个母题，为其都市变体和校园变体各取 1 个场景（共 8 个）
  └─ 打乱顺序，去掉所有标签（母题名、作品名、场景名）
  └─ 要求 LLM 将 8 个场景两两配对（同一母题的都市版 + 校园版 = 1 对）
  └─ 重复 3 次（每次随机抽取不同母题组合）
  └─ 衡量：配对准确率（正确配对 / 总配对数）
```

## 验收标准

### 相似度矩阵

| 指标 | 合格线 | 良好线 | 优秀线 |
|------|--------|--------|--------|
| 同母题内相似度 > 不同母题间相似度 | 至少 4/7 母题满足 | 至少 6/7 | 全部 7 个 |
| 同母题内相似度 > 同作品不同母题间相似度 | 至少 4/7 母题满足 | 至少 6/7 | 全部 7 个 |
| 同母题平均相似度 | > 0.6 | > 0.75 | > 0.85 |

### 母题链重构

| 指标 | 合格线 | 良好线 | 优秀线 |
|------|--------|--------|--------|
| ARI（调整兰德指数 vs 人工标注） | > 0.4 | > 0.7 | > 0.9 |
| NMI（归一化互信息，类别数不敏感） | > 0.5 | > 0.75 | > 0.9 |
| LLM 命名的母题与人工命名语义匹配 | > 3/7 匹配 | > 5/7 | > 6/7 |

### 配对盲测

| 指标 | 合格线 | 良好线 | 优秀线 |
|------|--------|--------|--------|
| 配对准确率（3 次平均） | > 60%（随机基线 25%） | > 80% | 100% |

### 输出产物

1. `results/similarity_matrix.json`（91 对相似度 + 分组统计）
2. `results/motif_chain_reconstruction.json`（LLM 自由聚类结果 + ARI 计算）
3. `results/blind_pairing_r1.json` ~ `results/blind_pairing_r3.json`（3 次配对盲测结果）

## 不依赖

纯 Python 脚本 + LLM API。依赖：
- p05 步骤 1 的产物（8 篇单篇母题提取结果中的 evidence 字段）
- `docs/gallery/fiction/motif.yaml`（作为 ground truth，但仅用于评估阶段）
- `docs/gallery/fiction/urban-romance/samples/` 和 `campus-romance/samples/`（场景原文）

## 结果预期

- **最容易识别的跨作品母题**："手势先于语言"。都市的毛巾和校园的纸巾共享完全一致的动作结构（发现需求→准备→不由自主地执行→回过神来→退缩/继续），LLM 几乎肯定能识别。
- **最难自动识别的母题**："旁观者的缺席与在场"。LLM 需要理解"缺席"本身也是一种叙事选择，而这需要对叙事结构进行元层次推理。可能需要引入对比概念（"一部有观众，一部没有"）来辅助判断。
- **关键发现**：如果同母题内相似度 > 同作品不同母题间相似度，说明母题相似度确实超越了作品边界的风格混淆，而非仅仅因为"同作品内的场景天然相似"。这一步对照是方法论的关键补充。
- **成本优化**：embedding 预筛选可将 LLM 调用从 91 次降至约 40-50 次（仅对模糊边界对做 LLM 判定），同时 embedding 结果本身也是一份独立的相似度基线数据。
- **如果失败**（ARI < 0.4，配对准确率接近随机）：说明母题过于抽象，或同一母题在不同作品中的变体差异过大，LLM 无法自动建立跨作品映射。此时 motif.yaml 的"共同提炼层"更适合作为人工辅助参考，而非自动化特征。
