# p03 — 风格可提取性实验

验证同一作者的文章是否共享可识别的风格特征，以及 LLM 能否从文本中提取、聚类和归因这些特征。

## 假设

1. 同一作者/同系列的文章存在可提取的共同风格特征
2. 这些特征的相似度 > 不同作者/不同系列的文章之间的相似度
3. LLM 可以基于风格描述将文章正确聚类和归因

## 实验步骤

```
步骤 1: 风格特征提取（每篇文章独立）
  └─ 对 A1-A5 和 B1-B3（共 8 篇），让 LLM 输出风格特征描述
  └─ 提示词约束：只描述模式，不说好坏，不评价内容
  └─ 输出格式：{
        "总括": "一句话风格定位",
        "句式特征": [...], 
        "用词偏好": [...],
        "情绪表达方式": [...],
        "细节使用模式": [...],
        "叙事视角特点": [...],
        "节奏感": [...]
      }

步骤 2: 风格聚类（blinded）
  └─ 将 8 篇的风格描述混合，去掉作者标签
  └─ 打乱顺序后交给 LLM，要求分成 2-3 组
  └─ 每组内文章风格应"显著相似"
  └─ 衡量标准：同作者文章落入同一组的比例

步骤 3: 风格归因
  └─ 取一篇新的文章（从 A 或 B 中留一法，选一篇不参与步骤 1）
  └─ 交给 LLM 上述 8 篇风格描述库 + 新文章正文
  └─ 要求判断新文章与哪一组风格最接近
  └─ 重复 8 次（每篇轮流做被归因对象）

步骤 4: 风格指南校准（可选）
  └─ 读取 style.md 中的 10 个美学关键词
  └─ 将步骤 1 提取的特征映射到这些关键词上
  └─ 衡量：LLM 自主提取的特征与 style.md 定义的吻合度
```

## 验收标准

### 聚类结果

| 指标 | 合格线 | 良好线 | 优秀线 |
|------|--------|--------|--------|
| 同作者聚类率 | > 50% | > 70% | > 90% |

### 归因结果

| 指标 | 合格线 | 良好线 | 优秀线 |
|------|--------|--------|--------|
| 归因准确率 | > 50%（8 篇中 4 篇正确） | > 75%（6 篇） | 100%（8 篇） |

随机基线：8 篇分 2 组，随机归因准确率 ≈ 50%。

### 输出产物

1. 8 篇文章各自的特征描述 JSON
2. 聚类结果矩阵（每篇文章所属组）
3. 归因混淆矩阵（实际作者 × 预测作者）
4. 风格指南吻合度报告（如执行了步骤 4）

## 不依赖

纯 Python 脚本 + LLM API。依赖案例库中的多作者数据。

## 结果预期

最可能的结果：
- 聚类准确率 60-80%（职场和校园有明显风格差异，但同一系列内部也有变异）
- 归因准确率 60-85%（B3 作为初稿可能风格不稳定，容易混淆）
- style.md 吻合度中等（LLM 提取的特征与人工定义的关键词部分重合）

如果聚类准确率低于 40%，说明"风格"在这个粒度上不可提取，StyleStore 的好文章积累策略需要重新设计——不能默认"同一作者/同一标签=同风格"。
